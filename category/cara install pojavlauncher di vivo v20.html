Cara Mengatasi Masalah Instalasi PojavLauncher
Solusi untuk Vivo V20 dengan Android 12 & FuntouchOS 10.5

Mengapa Terjadi Masalah?

Vivo V20 dengan Android 12 dan FuntouchOS 10.5 memiliki sistem keamanan yang ketat. Fitur "System Security" sering mencegah instalasi aplikasi seperti PojavLauncher yang bersumber dari luar Google Play Store.

Langkah-langkah Penyelesaian

1
Aktifkan Instalasi dari Sumber Tidak Dikenal
  • Buka Pengaturan di HP Vivo V20
  • Cari dan pilih Privasi atau Keamanan & Privasi
  • Temukan opsi Izinkan Instalasi dari Sumber Tidak Dikenal
  • Aktifkan toggle untuk browser atau file manager yang digunakan untuk mengunduh aplikasi
2
Nonaktifkan Sementara Keamanan Sistem FuntouchOS
  • Buka Pengaturan di HP Vivo
  • Cari dan pilih iManager atau Keamanan Sistem
  • Matikan sementara opsi Pemindaian Virus dan Perlindungan Sistem
  • Jika diminta konfirmasi, pilih "Nonaktifkan"
3
Unduh Versi PojavLauncher yang Kompatibel
  • Pastikan kamu mengunduh versi terbaru PojavLauncher (kompatibel dengan Android 12)
  • Unduh dari halaman resmi GitHub PojavLauncher atau situs tepercaya
  • Pilih versi yang sesuai dengan arsitektur perangkat (ARM64, ARM32, atau x86)
4
Gunakan File Manager Bawaan untuk Instalasi
  • Buka File Manager bawaan Vivo
  • Navigasi ke folder unduhan (biasanya di "Internal Storage > Download")
  • Ketuk file APK PojavLauncher yang sudah diunduh
  • Jika masih muncul peringatan keamanan, pilih "Tetap Instal" atau "Instal Saja"
5
Gunakan Mode Aman Vivo (Safe Mode)
  • Jika semua cara di atas belum berhasil, coba aktifkan Mode Aman
  • Matikan HP Vivo V20
  • Tekan dan tahan tombol volume bawah saat menghidupkan HP
  • Setelah masuk Mode Aman, coba instal PojavLauncher lagi
  • Setelah berhasil, restart HP untuk kembali ke mode normal
Penting!

Setelah berhasil menginstal PojavLauncher, jangan lupa untuk mengaktifkan kembali fitur keamanan di HP Vivo V20 untuk melindungi perangkat dari aplikasi berbahaya lainnya.

Tips Tambahan
  • Periksa Kompatibilitas: Pastikan PojavLauncher mendukung Android 12 dan spesifikasi HP Vivo V20
  • Hapus Cache: Jika instalasi gagal berulang kali, coba bersihkan cache dari aplikasi Pengaturan dan File Manager
  • Versi Alternatif: Coba versi PojavLauncher yang lebih lama jika versi terbaru tidak bisa diinstal
  • Izin Tambahan: Setelah instalasi, pastikan memberikan semua izin yang diperlukan oleh PojavLauncher untuk berfungsi optimal
